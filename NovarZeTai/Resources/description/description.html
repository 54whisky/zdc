【规格】 唑来膦酸。小瓶内装4mg唑来膦酸粉末（用5ml无 菌注射用水复溶）或4mg/5ml浓溶液，使用前应 进一步稀释。 【适应症】 • 与标准抗肿瘤药物治疗合用，用于治疗实体肿瘤 骨转移患者和多发性骨髓瘤患者的骨骼损害。 • 用于治疗恶性肿瘤引起的高钙血症(HCM)。 【剂量】 • 注射用唑来膦酸(择泰®4mg 粉针)先用5ml无菌 注射用水复溶。 • 在注射之前，还应将复溶的溶液或者 4mg/5ml 浓溶液(注射用唑来膦酸浓溶液)用 100ml 的 0.9%氯化钠溶液或 5%葡萄糖溶液进一步稀释。 制得的最终注射液应静脉输注给药，输注时间不 少于 15 分钟。本品不得与含钙或其它二价阳离 子的注射液(如乳酸林格液)配伍使用或接触，应 使用与其它药品分开的输液管进行单次静脉输 注。 • 不存在高钙血症的患者也应给予口服钙补充剂 500mg/日以及维生素 D 400IU/日。 • 对于“预防晚期恶性肿瘤骨转移患者的骨骼相关 事件”，本品的推荐剂量为 4mg 输注，每 3~4 周给药一次。在既往存在轻度至中度肾功能损害 的患者中，建议减量。 • 对于“治疗恶性肿瘤的高钙血症(HCM)”，本品的 推荐剂量为 4mg 单一剂量输注。轻度至中度肾 功能损害的患者不需要调整剂量。。 【禁忌】 • 孕妇。 • 哺乳期妇女。 • 对唑来膦酸、其它双膦酸盐或本品的任何辅料过 敏的患者。 【警告/注意事项】 • 给予本品前必须评价患者的情况，以确保他们处 于正常水化状态。 • 有发生心力衰竭危险的患者，应避免体内水分过 多。 • 监测标准的高钙血症相关代谢参数，例如血清钙 水平、磷和镁，尤其是血清肌酐水平。 • 考虑到双膦酸盐对于肾功能的潜在影响，而且严 重肾功能不全患者的临床安全性数据有限，因此 对于严重肾功能不全的患者不建议使用本品。 • 对于既往存在轻度至中度肾功能损害的患者，建 议减量使用本品。 • 对于需要重复使用本品的患者，应该在每次给药 前评估血清肌酐水平。如果出现肾功能减退的症 状，应停止用药。 • 由于严重肝功能不全患者的临床数据有限，因 此，对于此类患者没有特别的建议。 • 已经有报道颌骨坏死主要发生在使用双膦酸盐类 药物(包括本品)进行治疗的成年癌症患者中。上 市后经验和文献资料表明肿瘤类型(晚期乳腺 癌、多发性骨髓瘤)和牙齿状态(拔牙、牙周疾病 和包括义齿不适合在内的局部损伤)会导致颌骨 坏死发生率升高。因此，在本品治疗过程中，这 些患者应尽可能避免侵入性口腔治疗，保持良好 的口腔卫生，并且在患者使用双膦酸盐之前，考 虑进行预防性口腔检查或预防性牙科术。当患者 接受口腔治疗或有可预见的口腔手术时，应告知 主治医生。 • 在接受双膦酸盐(包括本品)治疗的患者中，报道 了非典型股骨和股骨干骨折，主要发生于长期使 用双膦酸盐治疗骨质疏松的患者中。此类患者应 该停用本品，并接受评估。本品治疗期间建议患 者报告任何股部、髋部或腹股沟疼痛。 • 在使用双膦酸盐类药物的患者中有报道严重和偶 然失能性骨骼、关节和/或肌肉疼痛。 • 接受本品(唑来膦酸)治疗的患者不应接受唑来膦 酸注射液(密固达®)治疗。本品也不应该与其他 双膦酸盐联用，因为这些联用的影响尚未知。 • 对乙酰水杨酸敏感的哮喘患者不应使用本品。 • 因为有观察到肾功能损害(包括肾衰竭)和血清电 解质紊乱(包括钙、磷和镁)，当剂量超过推荐剂 量时，应对患者进行密切监测。如临床上出现严 重的低钙血症，可根据临床指征输注葡萄糖酸 钙。 • 尚未建立本品对于儿童和青少年的安全性和有效 性。 • 接受本品治疗的患者曾报道过低钙血症。有报道 过继发于重度低钙血症的心律失常和神经系统不 良事件(癫痫发作、手足抽搐和麻木)。在某些情 况下，低钙血症可能危及生命。 • 建议注意本品与其他可引起低钙血症的药物一起 使用，因为他们可能产生协同效应导致严重低钙 血症。 • 使用本品治疗前应测定血钙浓度且必须纠正低钙 血症。患者应适当补充钙和维生素D。 • 育龄期妇女：建议育龄期妇女避免怀孕并告知用 本品治疗可能对胎儿造成的危害。 • 孕妇：参见禁忌。 • 哺乳期妇女：参见禁忌。
